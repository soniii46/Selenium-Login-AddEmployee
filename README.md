# OrangeHRM Automation

Automation scripts for OrangeHRM using **Selenium WebDriver** and **Java**.

This project includes test scripts for logging in and adding employees in the OrangeHRM application. It is designed to demonstrate basic automation skills for QA/test automation roles.

---

## **Features**

- **Login Automation**: Automates login functionality for OrangeHRM.
- **Add Employee Automation**: Automates adding new employee details in the system.
- Follows good practices like explicit waits and proper element locators.

---

## **Tech Stack**

- **Language**: Java  
- **Automation Tool**: Selenium WebDriver   
- **Build Tool**: Maven (optional)  
- **IDE**: Eclipse
